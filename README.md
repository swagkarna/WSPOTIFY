# WSPOTIFY

Brute-Force spotify accounts

## HOWTO

#### Please notice database_dump.txt must be sorted like: someone@gmail.com:password

         git clone https://github.com/wuseman/wspotify
         cd wspotify
         chmod +x wspotify
         ./wspotify <database_dump> 

## Requirements

   * Libspotify
   * Sconsify
   * Portaudio
